# Kaggle Upload Instructions

Upload this folder as the Kaggle dataset package:

```text
adaptive_data/
```

Recommended Kaggle title:

```text
NyayaSetu Legal Dialogues Multilingual Adaptive Dataset
```

Recommended Kaggle slug:

```text
nyayasetu-legal-dialogues-multilingual
```

Recommended description:

```text
NyayaSetu + ZamanatAI is a multilingual adaptive legal-access dataset for Indian civic and justice workflows. It includes WhatsApp-style legal aid queries, language labels, legal intents, entities, output document types, user feedback, correction events, adaptation history, and confidence-before/after metrics.

This dataset was built for the Adaption Labs Adaptive Data Track and demonstrates continuous learning through human feedback, corrections, multilingual expansion, and model-ready exports for legal AI agents, fine-tuning, and evaluation pipelines.
```

Files to include:
- adaptive_dataset.csv
- adaptive_dataset.jsonl
- dataset_audit.json
- adaptation_metrics.json
- language_statistics.csv
- feedback_statistics.csv
- correction_statistics.csv
- intent_statistics.csv
- adaptation_growth.csv
- accuracy_before_after.csv
- low_resource_language_coverage.csv
- README.md
- dataset_card.md
- demo_script.md

After uploading, add the Kaggle URL to the root project README and the Adaption submission form.
