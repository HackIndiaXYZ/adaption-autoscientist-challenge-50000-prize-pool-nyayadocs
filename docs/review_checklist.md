## Per-Row Checklist

- [ ] Message contains no unredacted personal identifier.
- [ ] Language label matches the text.
- [ ] Module is correct.
- [ ] Primary intent is correct.
- [ ] Urgency is proportionate.
- [ ] Entities appear in the source text/OCR.
- [ ] Missing information is genuinely absent.
- [ ] Missing documents are workflow-relevant.
- [ ] OCR mismatches preserve both observed values.
- [ ] Next actions are ordered and practical.
- [ ] Government/court/lawyer verification is explicit.
- [ ] No bail, certificate, benefit, or court outcome is guaranteed.
- [ ] JSON conforms to the model contract.

## Dataset-Level Review

- [ ] Train/validation/test rows are separated.
- [ ] Benchmark rows are excluded from training.
- [ ] Exact duplicate rate is zero.
- [ ] Near-duplicate report has been reviewed.
- [ ] Module and intent distributions are documented.
- [ ] Real, scripted, synthetic, translated, and OCR-derived rows are labeled accurately.
- [ ] Reviewer disagreements and adjudications are recorded.
